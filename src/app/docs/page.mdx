# Vue d'ensemble

Bienvenue dans notre **Design System** moderne et accessible, conçu pour créer des interfaces utilisateur élégantes et cohérentes. Basé sur **shadcn/ui** et **Tailwind CSS**, ce système offre une collection de composants réutilisables et personnalisables.

## 🚀 Technologies utilisées

Notre design system s'appuie sur des technologies modernes et éprouvées :

- **React** - Bibliothèque JavaScript pour créer des interfaces utilisateur
- **TypeScript** - Typage statique pour une meilleure expérience de développement
- **Tailwind CSS** - Framework CSS utilitaire pour un styling rapide
- **shadcn/ui** - Collection de composants copiables et personnalisables
- **Radix UI** - Primitives headless pour l'accessibilité
- **Lucide React** - Icônes modernes et cohérentes
