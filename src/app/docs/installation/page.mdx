## 1. Création d'un nouveau projet

Si vous partez de zéro, créez un nouveau projet Next.js :

<InstallTabs
  npm="npx create-next-app@latest mon-projet --typescript --tailwind --eslint --app"
  yarn="yarn create next-app mon-projet --typescript --tailwind --eslint --app"
  pnpm="pnpm create next-app mon-projet --typescript --tailwind --eslint --app"
  bun="bun create next-app mon-projet --typescript --tailwind --eslint --app"
/>

<CodeBlock language="bash">
    cd mon-projet
</CodeBlock>

## 2. Installation des dépendances

### Dépendances principales

Installez les packages essentiels pour le design system :

<InstallTabs
  npm="npm install tailwind-merge clsx class-variance-authority lucide-react"
  yarn="yarn add tailwind-merge clsx class-variance-authority lucide-react"
  pnpm="pnpm add tailwind-merge clsx class-variance-authority lucide-react"
  bun="bun add tailwind-merge clsx class-variance-authority lucide-react"
/>

### Dépendances Radix UI

Installez les primitives Radix UI nécessaires :

<InstallTabs
  npm="npm install @radix-ui/react-accordion @radix-ui/react-alert-dialog @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-dialog @radix-ui/react-slot"
  yarn="yarn add @radix-ui/react-accordion @radix-ui/react-alert-dialog @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-dialog @radix-ui/react-slot"
  pnpm="pnpm add @radix-ui/react-accordion @radix-ui/react-alert-dialog @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-dialog @radix-ui/react-slot"
  bun="bun add @radix-ui/react-accordion @radix-ui/react-alert-dialog @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-dialog @radix-ui/react-slot"
/>

### Dépendances pour les thèmes

Pour le support du mode sombre :

<InstallTabs
  npm="npm install next-themes"
  yarn="yarn add next-themes"
  pnpm="pnpm add next-themes"
  bun="bun add next-themes"
/>

## 3. Configuration Tailwind CSS

Mettez à jour votre fichier `tailwind.config.ts` avec la configuration complète pour shadcn/ui.

## 4. Variables CSS globales

Ajoutez les variables CSS dans votre fichier `globals.css` pour le support des thèmes clair et sombre.

## 5. Utilitaires

Créez le fichier `src/lib/utils.ts` avec la fonction `cn` pour fusionner les classes CSS.

## 6. Configuration des thèmes

### Provider de thème

Créez le composant ThemeProvider pour gérer les thèmes.

### Intégration dans layout

Intégrez le ThemeProvider dans votre layout principal.
