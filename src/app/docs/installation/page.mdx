# Installation

Pour installer le design system dans votre projet, utilisez l'une des commandes ci-dessous selon votre gestionnaire de paquets préféré :

<InstallTabs
  npm="npm install mon-design-system"
  yarn="yarn add mon-design-system"
  pnpm="pnpm add mon-design-system"
  bun="bun add mon-design-system"
/>

## Prérequis

- Node.js >= 18
- Un gestionnaire de paquets (npm, yarn, pnpm ou bun)

## Exemple d'import

```tsx
import { Button } from "mon-design-system";

function App() {
  return <Button>Mon bouton</Button>;
}
```

C'est tout ! Vous pouvez maintenant utiliser les composants du design system dans votre application.
