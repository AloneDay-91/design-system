## Anatomie

L'accordion est composé de plusieurs parties qui fonctionnent ensemble :

```tsx
import {
  Accordion,      // Conteneur principal
  AccordionItem,  // Élément individuel
  AccordionHeader,// En-tête de l'élément
  AccordionTrigger, // Bouton pour ouvrir/fermer
  AccordionPanel  // Contenu pliable
} from "@/design-system";

<Accordion>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger />
    </AccordionHeader>
    <AccordionPanel />
  </AccordionItem>
</Accordion>
```
