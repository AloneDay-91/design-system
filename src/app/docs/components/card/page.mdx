## Vue d'ensemble

Le composant Card fournit une structure de base pour organiser le contenu en sections visuellement distinctes. Il se compose de plusieurs sous-composants qui permettent une organisation flexible du contenu.

## Bonnes pratiques

### ✅ À faire
- Utilisez des titres clairs et descriptifs
- Gardez le contenu organisé et lisible
- Utilisez les grilles pour organiser plusieurs cartes

### ❌ À éviter
- N'empilez pas trop d'informations dans une seule carte
- Évitez les cartes trop grandes ou trop petites
- Ne négligez pas l'espacement entre les éléments
